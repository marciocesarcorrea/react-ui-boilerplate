## Prerequisites

- node >=10.15.x
- yarn >=1.20.0

## Install

```sh
yarn install
```

## Usage

```sh
yarn run start
```

## Run tests

```sh
yarn run test
```

## Run build

```sh
yarn run build
```

## Run build with local server http

```sh
yarn run build:serve
```
